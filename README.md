# Photoshopスクリプト all layersets merge

- すべてのグループをグループごとに結合するスクリプト（コマンドEを連続する）
- 特定のグループを除外したりはできません
- ファイルの保存はしません
- グループ化されていないレイヤーは無視されます（現状維持）

*大量のグループをそれぞれに結合したい時に便利です*

**アートボードをご利用の場合はグループごとではなく、アートボードごとに結合されます**
